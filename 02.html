2.1- ¿qué es una url absoluta?

Una URL absoluta contiene el protocolo http:// o https, el nombre del dominio y la ruta que es necesario. Y se debe usar un URL absolut cuando apunta al documento.

2.2- ¿qué es una url relativa?

las URL relativas describen un archivo relativo actual al documento. Estas mismas se pueden usar cuando se une con otro documento
